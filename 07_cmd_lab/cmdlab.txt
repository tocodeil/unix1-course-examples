2. View the file /etc/passwd on your machine with the command "cat /etc/passwd".
 Then read about the file in "man 5 passwd". What information is held in the file? What are the fields in each line?

it's a password file. holds per each user one line and 7 fields :
login name 
optional encrypted password
numerical user ID
user name or comment field
user home directory
optional user interpreter
